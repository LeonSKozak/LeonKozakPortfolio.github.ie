Power BI Project
G20 Energy and Sustainability

The goal of this project was to create a Power BI Report about a data set. This report was created in a group project with Anika Stammen, Leonie RÃ¶skamp, Lasse Willen and myself.

G20 Data Set

The Screenshots of the Report can also be found here in the folder Project_Codes: Data Science Portfolio on GitHub!

This Power BI Report visualized the data on the G20 energy and sustainability report. The altered data set can be found below.

The aim of the report was to present the data in a clear and interesting way in order to highlight important findings. The data relates to the period from 1990 to 2023. Each page of the Power BI report was designed to be interactive so that, for example, different years or countries could be selected. The Power BI Report was also presented as part of the MT412 module at DCU using a PowerPoint presentation. Both, the brief Power Point presentation and screenshots of the report, can be found below.
