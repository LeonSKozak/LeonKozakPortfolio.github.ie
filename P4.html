Python Project 4
Car Price Prediction and Clustering

This project was created together with my fellow students Lasse Willen and Vincent Stilz. The data set can be found under the following link:

Car Sales Data Set

The Python File can also be found here in the folder Project_Codes: Data Science Portfolio on GitHub!

The data set contains valuable information on the specific cars. These include the model, the year, the model and a specific trim, its type, meaning SUV, Sedan etc., and the transmission type. Furthermore, the data set also contains data on the condition (1= worst, 49=best), the odometer and the selling price among others. The data set can be found under the link above or below: In this project we first did a supervised Linear Regression Model, for which we had to split the data, process and model it and then train it, before we were able to make our firat prediction. We then did a Random Forrest Model, which showed better performance when it came to MSE prediction accuracy. Next, we did an Unsupervised Model (K-Means) to cluster the data, using the elbow method. The elbow method is a technique used to determine the optimal number of k-mean clusters. Furthermore, dimensionality reduction has been performed to simplify the data for processing, without losing information. This was helpful for the visualization that was performed. First, basic visualization was performed, then, centroids were added and interactive plots and 3D visualization was performed.
