Python Project 1
Apple Features

In this project, I worked on a data set about the quality and different features of apples. Apple Features Data Set

The Python File can also be found here in the folder Project_Codes: Data Science Portfolio on GitHub!

Below, you can take a look at the data set.

For this project, first data cleaning has been done. Although, it must be said that the data was very clean to begin with. Next, the data distribution was checked - here, no significant outliers where detected. In the next steps, interesting stories where found and visualized, that could be found within the data set. Also, I tried to find out which apple features are most important for the apple to score a high or low quality. This has been calculated using both correlations, and a random forrest model.

The Notebook can be found here: Python Project 1 Notebook.
